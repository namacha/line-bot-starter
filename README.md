# line-bot-starter

A starter kit for Line Bot


## Making bot
1. Open `settings.py` and edit `LINE_CHANNEL_ACCESS_TOKEN` and `LINE_CHANNEL_SECRET`
2. Write bot logic in `app.py`
3. Deploy. (I recommend to use Heroku)
